# KVIK Report Download Guide

## Portal Access
**URL**: https://www.kviconline.gov.in/pmegpeportal/pmegpmr3/reportHomePage.jsp

## Date Range
- **From Date**: 01-APR-2025
- **To Date**: 17-MAR-2026

## Steps to Download

1. **Open the Portal**
   - Go to: https://www.kviconline.gov.in/pmegpeportal/pmegpmr3/reportHomePage.jsp

2. **Fill in Date Fields**
   - From Date: `01-APR-2025`
   - To Date: `17-MAR-2026`

3. **Select Report Type** (if applicable)
   - Look for any dropdown filters (state, district, etc.)

4. **Click Download/Submit**
   - Click the "Download" or "Generate Report" button

5. **Save File**
   - Save the Excel file to: `E:\kadhi\PMEGP_project\kvik_downloads\`

## Alternative: Using Browser Console (for API calls)

If the portal has an API endpoint, you can:

1. Open DevTools (F12)
2. Go to Network tab
3. Use the form and capture the API request
4. Copy the request details to automate it

## File Location
Downloaded files should go to: `E:\kadhi\PMEGP_project\kvik_downloads\`

## After Download
Once downloaded, you can:
- Compare with existing format templates
- Align data to match your standard Excel structure
- Use pandas/openpyxl to transform and validate the data
